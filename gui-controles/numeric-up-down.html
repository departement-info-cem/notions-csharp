NumericUpDown

Un contrôle NumericUpDown est un champ numérique dans lequel des flèches permettent d'augmenter ou diminuer la valeur affichée.

PropriétéTypeDescription

(Name)		Indique le nom utilisé dans le code pour identifier l’objet. Préfixe : num Exemple: numCoutProduit

Enabled		boolObtient ou définit une valeur indiquant si le contrôle est activé ou non.

Visible		boolObtient ou définit une valeur indiquant si le contrôle est visible ou non

DecimalPlaces		intIndique le nombre de décimales à afficher.

Increment		decimalIndique l'incrémentation ou la décrémentation à chaque clic sur le bouton.

Maximum		decimalIndique la valeur maximale possible pour ce contrôle.

Minimum		decimalIndique la valeur minimale possible pour ce contrôle.

ReadOnly		boolIndique si la zone d'édition est en lecture seule. Si true alors l'incrément doit être 0.

TextAlign		Détermine la position du texte dans la zone d'édition. Les nombres doivent être alignés à droite.

Value		decimalIndique la valeur actuelle placée dans le contrôle.
ÉvénementDescription

ValueChanged		Se produit lorsque la valeur du contrôle change.
